# 项目简介

## 核心功能
- 基于YOLOv8的河道漂浮物检测系统
- 支持图片、视频及摄像头进行检测
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 可选择检测模型
- 界面可实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图

## 适用场景
- 河道漂浮物监测
- 水环境保护
- 水质管理
- 水域安全监测
- 环境保护监测

## 技术特点
- 使用Python3.9开发
- 集成opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整源码和训练好的模型
- 包含9000张数据集，专注于'floating'一类目标检测
- 配置完整环境后可直接使用
- 提供UI界面源文件和图标
- 支持多种输入源（图片、视频、摄像头）
